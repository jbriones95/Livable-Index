# Livable Index

Interactive Littleton, CO livability explorer built with React + Vite and Leaflet.

Features
- Choropleth overlay of synthesized livability scores for named zones
- Click a zone to view composite score and per-dimension breakdown
- Deployable to GitHub Pages

Local
- npm install
- npm run dev -> http://localhost:5173/

Deploy (GitHub Pages)
- See DEPLOY.md for exact commands. The project includes vite.config.js and gh-pages scripts.
